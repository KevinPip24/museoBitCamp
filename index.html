Benvenuti a BitCamp Museo

Benvenuti al progetto di gestione del database per il museo. Questo sistema è stato progettato per aiutare il museo a gestire e organizzare le informazioni sui suoi reperti, mostre, artisti e visitatori.

Esplora


Descrizione del Database

Il nostro database comprende diverse tabelle, tra cui:

  • Opere d'arte
  • Artisti
  • Eventi
  • Visite

Le tabelle sono collegate tra loro tramite relazioni chiave primarie e chiave esterne, fornendo una struttura organizzata per l'archiviazione e il recupero dei dati.



Funzionalità del Progetto

Il nostro progetto consente agli utenti di:

  • Ricerca opere d'arte
  • Visualizzare gli eventi in corso
  • Registrare le visite

Tecnologie Utilizzate

Abbiamo utilizzato le seguenti tecnologie:

  • Linguaggi di programmazione: HTML, CSS, JavaScript
  • Framework: Bootstrap
  • Database: MySql
